## math class (simple.h)

### basic math 

```cpp
    add =
    simple::Add(x , y)
    {x + y}
    
    deduct = 
    simple::deduct(x , y)
    {x - y}
    
    times =
    simple::times(x , y)
    {x * y}
    
    devide = 
    simple::devide(x , y)
    {x / y}
```

### array math

```cpp
    sum = 
    simple::sum();
    
    lowest Number = 
    simple::lowestNum();
    
    biggest Number = 
    simple::biggestNum();
```

### square root

```cpp
    square =
```
